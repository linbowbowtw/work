    行程介紹

    // 宇宙星光 燈飾庭園
    2017年10月開幕的星光庭院主題公園用燈光呈現出十座宇宙相關內容，位於入口處的「太空站」透過媒體藝術打造生動燈光饗宴；以燈泡妝點而成的「花園」裡，如星光般閃爍的花朵們隨徐風搖曳生姿。而全韓國最長的101公尺長隧道「銀河隧道101」將讓人有種漫步於銀河之中的奇妙感受；結合燈光與音樂的燈光秀「星光交響樂團」則如真正樂團演奏般磅礡精彩。

    // 河溝生態公園
    因為擁有優美的風景與良好的生態環境，被列為始興市的1級生態環境保護區，2012年2月更被指定為國家海洋濕地保護地區。因韓劇<男朋友>拍攝地,近期更成為許多韓國人前往的休閒區。

    // 利川大米麵包DIY
    白米盛產於中部的京畿道，多條河流在此匯聚成漢江，水源充沛，適合種米。不過種出來的米質素亦有高低，當中以西部利川地區產的最優質，因為當地地勢較高，沒有高山遮擋，陽光充足，晚上低溫，稻米會生產更多糖分禦寒。韓國政府曾做過化驗，發現這兒的水質是全京畿道最佳，含豐富礦物質，種出來的米格外清甜煙韌，粒粒分明，米香悠長，質素不比越光、純情等日本高級米遜色。為了提升利川米味道，當地更實施環保耕種法，在農地飼養鴨子、蝸牛，讓牠們吃掉雜草。其間不下任何農藥，全用天然肥料。每塊稻田二至三年才產一造米，中間有九至十個月時間讓土地休養生息。

    因為擁有優美的風景與良好的生態環境，被列為始興市的1級生態環境保護區，2012年2月更被指定為國家海洋濕地保護地區。因韓劇<男朋友>拍攝地,近期更成為許多韓國人前往的休閒區。

    // 利川大米麵包DIY
    白米盛產於中部的京畿道，多條河流在此匯聚成漢江，水源充沛，適合種米。不過種出來的米質素亦有高低，當中以西部利川地區產的最優質，因為當地地勢較高，沒有高山遮擋，陽光充足，晚上低溫，稻米會生產更多糖分禦寒。韓國政府曾做過化驗，發現這兒的水質是全京畿道最佳，含豐富礦物質，種出來的米格外清甜煙韌，粒粒分明，米香悠長，質素不比越光、純情等日本高級米遜色。為了提升利川米味道，當地更實施環保耕種法，在農地飼養鴨子、蝸牛，讓牠們吃掉雜草。其間不下任何農藥，全用天然肥料。每塊稻田二至三年才產一造米，中間有九至十個月時間讓土地休養生息。
